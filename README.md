<h3 align="center">_ SETUP _</h3>

&nbsp;

### 🚀 How to install?

_No. There are no **ALL-IN-ONE** install script._

👇

&nbsp;

#### 🔧 Configs

If you are familar with _dotfiles_ (`~/.config`, `~/.local/bin`...), just clone the repo:

```bash
git clone https://github.com/ForeverWeLearn/niri-dotfiles.git
```

And copy them from `home/.config` to your `~/.config`:

&nbsp;

##### 🔥 niri

`home/.config/niri` -> `~/.config/niri`

&nbsp;

##### 🦉 Noctalia

`home/.config/noctalia` -> `~/.config/noctalia`

&nbsp;

##### 😽 kitty

`home/.config/kitty` -> `~/.config/kitty`

&nbsp;
